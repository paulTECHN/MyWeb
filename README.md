# IBIRUNGA TOUR (HTML-Only Website)

## Project Description
IBIRUNGA TOUR is a simple multi-page website created using **only HTML**.  
It introduces the **Virunga (Ibirunga) Mountains** and **Mountain Gorillas** in Rwanda.  
This project was done as a practical activity to practice core HTML elements and website structure.

## Tools Used
- **Figma** (for mockup design)
- **HTML only** (no CSS, no JavaScript)
- **GitHub** (for submission)

## Pages Included
- `index.html` — Home page (introduction and navigation)
- `about.html` — About Ibirunga and gorillas
- `services.html` — Tours and packages (lists + table)
- `media.html` — Images, audio, and video
- `form.html` — Booking form
- `contact.html` — Contact details and embedded map (iframe)

## How to Open the Website
1. Download or clone the repository
2. Open `index.html` in a browser
3. Use the navigation menu to visit other pages
